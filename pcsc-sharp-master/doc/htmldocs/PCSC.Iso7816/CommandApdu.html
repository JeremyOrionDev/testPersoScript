CommandApdu Class

A Command Application Protocol Data Unit (APDU), defined by the ISO/IEC 7816 standards

Syntax

public class CommandApdu : Apdu, ICloneable

Remarks

A PCSC.Iso7816.CommandApdu can be build using one of the following PCSC.Iso7816.IsoCase:

Case APDU structure
1 CLA INS P1 P2
2 CLA INS P1 P2 Le
3 CLA INS P1 P2 Lc Data
4 CLA INS P1 P2 Lc Data Le

See the documentation for PCSC.Iso7816.IsoCase for more information.
