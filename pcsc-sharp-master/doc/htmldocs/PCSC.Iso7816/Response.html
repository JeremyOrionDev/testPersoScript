Response Class

An aggregation of PCSC.Iso7816.ResponseApdu instances.

Syntax

public class Response : IEnumerable<ResponseApdu>

Remarks

When using IIsoReader.Transmit(CommandApdu) the result can consist of one or more PCSC.Iso7816.ResponseApdu. If the PCSC.Iso7816.IIsoReader receives a SW1=0x61 status word, it will automatically transmit a GET RESPONSE command to reader (after waiting IIsoReader.RetransmitWaitTime ms) to catch all remaining PCSC.Iso7816.ResponseApdu.
